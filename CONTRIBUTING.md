# Contributing to Electronic Tasbeeh App

First of all, thank you for considering contributing to the Electronic Tasbeeh App! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs

This section guides you through submitting a bug report for the Electronic Tasbeeh App. Following these guidelines helps maintainers and the community understand your report, reproduce the behavior, and find related reports.

- **Check if the bug has already been reported.**
- **Use a clear and descriptive title.**
- **Describe the steps to reproduce the issue.**

### Suggesting Enhancements

This section guides you through submitting an enhancement suggestion for the Electronic Tasbeeh App, including completely new features and minor improvements to existing functionality.

- **Use a clear and descriptive title.**
- **Provide a detailed description of the suggested enhancement.**
- **Explain why this enhancement would be useful.**

### Pull Requests

1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure your code lints and builds correctly.
4.  Issue that pull request!

## Development Setup

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npx expo start` to start the development server.

## Coding Style

*   Keep code clean and readable.
*   Comment complex logic.
*   Follow React Native best practices.

Thank you for your contributions!
